# Pneumara

Pneumara is a calming, AI-powered mental wellness companion built with Next.js. It combines compassionate conversational support, mood tracking, self-care tools, and crisis-aware guidance into a single, focused experience designed to help users reflect, reset, and stay connected to their well-being.

## Problem It Solves

Many wellness apps address only one part of the mental health journey. Users often need a place to talk through how they feel, track emotional patterns over time, and access immediate grounding tools without switching between different products. Pneumara addresses that gap by bringing supportive conversation, reflective self-check-ins, and practical self-care resources into one lightweight interface.

The application is designed to provide:

- A low-friction space for emotional expression and reflection
- Simple mood logging that makes patterns easier to notice
- Guided self-care features that support regulation and calm
- Crisis-aware messaging and resources when urgent support may be needed

## Features

- AI mental health chat powered by Groq and the AI SDK
- Compassionate, scope-limited assistant behavior focused on wellness, mindfulness, and emotional support
- Crisis keyword detection with emergency resource prompts
- Mood selection onboarding and conversation starters
- Mood tracker with local persistence, streaks, summaries, and data visualizations
- Guided 4-4-4 breathing exercise
- Wellness quotes and practical self-care tips
- Ambient background sounds for relaxation
- Google sign-in plus guest access
- Theme switching and responsive mobile-first navigation
